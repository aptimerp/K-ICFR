## 금강공업 ERP 핵심 용어

### 회계·전표 영역
| 용어 | 영문 | 정의 | 예시 |
|-----|------|------|------|
| **전표** | Voucher / Slip | 회계 거래를 기록하는 단위 문서 | 매출 전표, 구매 전표, 지급 전표 |
| **전표 마감** | Period Close | 월/분기 말 전표 확정 (이후 수정 불가) | 월말 마감 후 수정 요청 불가 |
| **계정과목** | GL Account | 회계 자산·부채·수익·비용 분류 체계 | 3110(선급금), 4110(상품매출) |
| **부서코드** | Department Code | 조직 단위별 코드 | SAL(판매), MFG(제조) |

### 수불 영역
| 용어 | 영문 | 정의 | 예시 |
|-----|------|------|------|
| **수불** | Inventory Mgmt | 입출고·재고 관리 | 원재료 입고 → 제조 → 완성품 출고 |
| **입고** | Receipt | 구매 물품·반제품 입고 기록 | PO 기반 입고, 반입 입고 |
| **출고** | Issue | 재고에서 물품 꺼내기 (제조·판매용) | 생산 출고, 판매 출고 |
| **재고** | Inventory Balance | 창고에 남은 물품 수량·가액 | 현재 재고 5,000개 @ 10,000원 |
| **언양** | Unyang Factory | 언양 공장 (수불 독립 운용) | 별도 수불 시스템, ERP와 분리 |

## 금강공업 특수 규칙

### 전표 관련
1. **월말 마감 후 수정 금지** — 재무팀이 마감한 전표는 감시자 권한 필요
2. **환율 적용** — 외화 거래는 거래일 환율 자동 적용
3. **선급금 관리** — 선급금 입금 → 수주 매칭 → 자동 상계

### 수불 관련
1. **언양 공장 분리** — 본사·언양 별도 재고 시스템 → 월 1회 대사
2. **FIFO 원칙** — 평균원가법 미사용 (FIFO 엄격 적용)
3. **Zero-stock 점검** — 매월 2일 재고 실사 + 전표화

### 내부통제
1. **3자 승인** — 결재 금액 ₩10M 이상: 과장 → 부장 → 이사 순차 승인
2. **시간차 관리** — 매입·매출 분개 시간 로깅 (감시용)
3. **감시자 권한** — CFO·감시 담당자만 과거 전표 수정 가능
