### 데이터 흐름 (확정)
```
ERP → 운영관리 > 부서·사원 관리 (원본 수신)
              ↓ Supabase departments / users
시스템관리 > 사용자·권한 (계정 설정·역할 배정)
```

└── seeds/
    └── rcm_controls.sql         ← RCM 402건
lib/supabase.ts                  ← 클라이언트 + TS 타입
.env.local.example               ← 환경변수 템플릿
